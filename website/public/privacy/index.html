Ext Helper Privacy Policy

Ext Helper is a browser extension manager for Chrome, Firefox, and Edge. This page explains what the extension stores, what it does not collect, and why the requested permissions are needed.

Last updated: April 19, 2026

Overview

Ext Helper does not collect, transmit, or share any user data with external servers.

Data Collection

  • No personal information is collected.
  • No browsing history is transmitted to any remote service.
  • No analytics, tracking pixels, or third-party sharing are used.

Data Stored Locally

The extension stores the following data locally on the device only:

  • Extension groups created by the user
  • Automation rules configured by the user
  • UI preferences such as theme and view mode
  • Bisect session state used during extension conflict debugging

Permissions Explanation

Permission Purpose
management Read and control installed browser extensions.
storage Save groups, rules, and preferences on the local device.
tabs Read the active tab URL so domain-based rules can be evaluated.
alarms Schedule time-based automation rules.
<all_urls> Match the active tab URL against user-defined domain rules only.
