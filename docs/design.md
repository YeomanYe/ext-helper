# UI/UX 设计文档

## 1. 设计原则

### 1.1 核心原则

1. **清晰优先** - 信息层级明确，用户一眼能找到目标
2. **高效操作** - 减少点击次数，常用操作一步直达
3. **一致性** - 统一的视觉语言和交互模式
4. **可访问性** - 符合WCAG 2.1 AA标准

### 1.2 设计价值观

```
┌─────────────────────────────────────────────────────────────┐
│                     ExtHelper Design                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   高效 ─── 快速找到目标，一键完成操作                        │
│           ↓                                                │
│   简洁 ─── 界面干净整洁，没有多余元素                        │
│           ↓                                                │
│   直观 ─── 操作结果可预期，反馈及时                          │
│           ↓                                                │
│   兼容 ─── 适配多种浏览器，保持体验一致                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 4. 交互设计

### 4.1 动画规范

| 动画 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| fade-in | 150ms | ease-out | 显示元素 |
| fade-out | 100ms | ease-in | 隐藏元素 |
| slide-down | 200ms | ease-out | 展开分组 |
| slide-up | 150ms | ease-in | 收起分组 |
| scale | 150ms | ease-out | 按钮反馈 |
| toggle | 200ms | ease-in-out | 开关切换 |

### 4.2 过渡效果

```css
/* 基础过渡 */
.transition-all {
  transition: all 150ms ease-out;
}

/* 特定属性 */
.transition-colors {
  transition: color 150ms ease-out,
              background-color 150ms ease-out,
              border-color 150ms ease-out;
}

.transition-transform {
  transition: transform 150ms ease-out;
}
```

### 4.3 拖拽交互

```
┌─────────────────────────────────────────────────────────┐
│  拖拽前                    拖拽中                      │
│                                                         │
│  ┌─────────────────┐         ┌─────────────────┐        │
│  │ Ext-A           │         │ Ext-A           │        │
│  │ [📷] Name       │  ───→   │ [📷] Name       │ 拖拽   │
│  └─────────────────┘         │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓│ 元素   │
│                             └─────────────────┘        │
│                                                         │
│  ┌─────────────────┐         ┌─────────────────┐        │
│  │ Ext-B           │         │ ░░░░░░░░░░░░░░░░░│ 放置   │
│  │ [📷] Name       │         │ 放置区域        │ 目标   │
│  └─────────────────┘         └─────────────────┘        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**视觉反馈**：
- 拖拽元素：半透明 + 阴影 + 旋转2deg
- 放置区域：虚线边框 + 背景高亮
- 拖拽插入点：指示线

### 4.4 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + F` | 聚焦搜索框 |
| `↑/↓` | 选择扩展 |
| `Enter` | 切换启用状态 |
| `←` | 展开分组 |

| `Delete` | 从分组移除 |
