# UI/UX 设计文档

## 1. 设计原则

### 1.1 核心原则

1. **赛博朋克美学** - 暗色主题、霓虹光效、终端风格
2. **高效操作** - 减少点击次数，右键菜单+下拉菜单整合功能
3. **即时反馈** - 乐观更新，操作立即响应
4. **信息密度** - 多视图模式适配不同场景

### 1.2 设计价值观

```
┌─────────────────────────────────────────────────────────────┐
│                     ExtHelper Punk Design                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   赛博朋克 ─── 暗色主题，霓虹光效，终端字体                │
│              ↓                                             │
│   高效 ─── 三种视图，右键菜单，Actions 下拉               │
│              ↓                                             │
│   直观 ─── 分组标签直接操作，Tab 切换                      │
│              ↓                                             │
│   强大 ─── Bisect 调试，规则自动化，撤销/重做              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 3. 组件设计

### 3.1 扩展卡片 - 紧凑模式（Grid View）

```
┌─────────┐
│   📷   │
│    ●    │
│  Name   │
└─────────┘
```

**设计细节**：
- 正方形卡片（aspect-square）
- CSS grid: `grid-cols-[repeat(auto-fill,minmax(80px,1fr))]`
- 图标（40x40）+ 状态指示点 + 名称（截断14字符）
- 点击卡片切换启用/禁用
- 禁用时降低透明度

### 3.2 扩展卡片 - 卡片模式（Card/List View）

```
┌─────────────────────────────────────────────────────────────┐
│  ┌────┐  Extension Name                            [1/0]  │
│  │ 📷 │  Description text here...                          │
│  └──●─┘  v1.2.3                                           │
└─────────────────────────────────────────────────────────────┘
```

**设计细节**：
- 水平布局，单列，最小高度 60px
- 图标（40x40）+ 状态点 + 名称 + 描述 + 版本 + 二进制开关
- 右键显示操作菜单

### 3.3 扩展卡片 - 详情模式（Detail View）

```
┌─────────────────────────────────────────────────────────────┐
│  ┌──────┐  EXTENSION NAME              v1.2.3             │
│  │  📷  │                               [1/0]             │
│  └────●─┘  [ACTIVE]  development                          │
│                                                             │
│  Description text here, full description visible.          │
│                                                             │
│  ┌─ PERMISSIONS (5) ──────────────────────────────────┐   │
│  │ [perm1] [perm2] [perm3] [perm4] [perm5]            │   │
│  └────────────────────────────────────────────────────┘   │
│                                        [OPTIONS] [REMOVE] │
└─────────────────────────────────────────────────────────────┘
```

**设计细节**：
- 全宽卡片
- 图标（56x56）+ 名称 + 版本 + 开关 + 状态标签 + 安装类型
- 完整描述
- 权限列表（最多显示6个，超出显示 "+n more"）
- 底部操作按钮（OPTIONS、REMOVE）

### 3.4 分组标签（GroupChips）

```
┌─────────────────────────────────────────────────────────────┐
│  [● 工作 [4] ⏻]  [● 开发工具 [6] ⏻]  [+]                │
└─────────────────────────────────────────────────────────────┘
```

**设计细节**：
- 水平 flex-wrap 布局
- 颜色点 + 分组名 + 数量 + 电源按钮
- 点击标签 → 打开分组弹窗
- 点击电源按钮 → 一键启用/禁用全组
- "+" 标签创建新分组

### 3.5 右键菜单

```
┌─────────────────────┐
│ ⚡ ENABLE/DISABLE   │
├─────────────────────┤
│ ⚙ OPTIONS          │  ← 仅扩展有设置页时显示
├─────────────────────┤
│ ℹ DETAILS          │
├─────────────────────┤
│ 🗑 REMOVE           │
└─────────────────────┘
```

**设计细节**：
- 4个菜单项（OPTIONS 条件显示）
- 使用 Portal 渲染到 body
- 自动计算位置避免溢出
- 点击外部关闭

### 3.6 Actions 下拉菜单

```
┌────────────────┐
│ Start Bisect   │
│ Enable All     │
│ Disable All    │
│ Undo [3]       │
│ Redo [1]       │
└────────────────┘
```

Bisect 激活时显示 Bisect Good / Bisect Bad / Cancel Bisect

### 3.7 BisectBanner

```
┌─────────────────────────────────────────────────────────────┐
│  BISECT STEP 3                          [Good] [Bad] [Cancel]│
│  Good = issue disappeared, Bad = issue still present        │
│  Candidates 4 · Testing 2                                   │
└─────────────────────────────────────────────────────────────┘
```

Resolved 状态：
```
┌─────────────────────────────────────────────────────────────┐
│  BISECT RESOLVED                      [Restore Original]    │
│  Suspect: Dark Reader                                       │
└─────────────────────────────────────────────────────────────┘
```

## 4. 交互设计

### 4.1 核心交互

| 操作 | 触发方式 | 结果 |
|------|----------|------|
| 切换启用/禁用 | 点击开关（Card/Detail）/ 点击卡片（Compact） | 立即切换状态（乐观更新） |
| 打开分组弹窗 | 点击分组标签 | 打开编辑弹窗 |
| 启用/禁用全组 | 点击分组电源按钮 | 一键切换分组内所有扩展 |
| 更多操作 | 右键点击卡片 | 显示4项菜单 |
| 查看详情 | 右键 → DETAILS | 打开详情弹窗 |
| 切换视图 | Header 按钮（GRID/CARD/DETAIL） | 切换三种模式 |
| 切换标签 | TabBar: EXTENSIONS / RULES | 切换主内容 |
| 批量操作 | ACTIONS 下拉 | Enable All/Disable All/Bisect/Undo/Redo |
| 关闭弹窗 | ESC / 点击遮罩 | 关闭弹窗 |
| Bisect | BisectBanner: Good/Bad | 推进二分搜索 |

### 4.2 动画规范

| 动画 | 时长 | 用途 |
|------|------|------|
| transition-all | 200ms | 通用状态变化 |
| transition-colors | 150ms | 颜色变化（hover） |
| animate-pulse-neon | - | 状态指示点脉冲 |
| animate-scanline | - | Header 扫描线装饰 |
| backdrop-blur-sm | - | 弹窗背景模糊 |

## 5. 响应式设计

### 5.1 Popup尺寸

```
宽度: 不固定（由浏览器扩展弹窗决定）
高度: 600px (固定, flex h-[600px])
```

### 5.2 网格布局

- **紧凑模式**：CSS grid `auto-fill, minmax(80px, 1fr)`，自动填充列
- **卡片模式**：单列 `grid-cols-1 gap-2`
- **详情模式**：单列 `flex flex-col gap-2`

## 6. 视觉设计

### 6.1 色彩系统（Punk 主题）

| 名称 | 用途 | Hex |
|------|------|-----|
| Primary | 主色/边框/按钮 | `#7C3AED` |
| Secondary | 次级文字/hover | `#A78BFA` |
| CTA | 危险操作/警告 | `#F43F5E` |
| Accent | 高亮/链接/图标 | `#22D3EE` |
| Success | 启用状态/在线 | `#10B981` |
| Warning | Bisect 警告 | `#FBBF24` |
| Background | 主背景 | `#0F0F23` |
| Background Alt | 卡片背景 | `#1A1A2E` |
| Text Primary | 主文字 | `#E2E8F0` |
| Text Secondary | 次文字 | `#94A3B8` |
| Text Muted | 弱化文字 | `#64748B` |
| Neon Cyan | Logo/装饰 | `#00FFFF` |
| Neon Pink | 故障效果 | `#FF00FF` |

### 6.2 字体系统

| 字体 | 类名 | 用途 |
|------|------|------|
| Press Start 2P | `font-punk-heading` | 标题、按钮、标签 |
| VT323 | `font-punk-body` | 正文、描述 |
| Fira Code | `font-punk-code` | 版本号、代码 |

### 6.3 间距系统

```
按钮间距: gap-1 ~ gap-2
卡片内边距: p-2.5 ~ p-3
区域间距: gap-2 ~ gap-3
边框: border / border-2
```
