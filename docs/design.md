# UI/UX 设计文档

## 1. 设计原则

### 1.1 核心原则

1. **简洁优先** - 只显示必要信息，其他通过交互获取
2. **高效操作** - 减少点击次数，右键菜单整合所有功能
3. **一致性** - 统一的视觉语言和交互模式
4. **可访问性** - 符合WCAG 2.1 AA标准

### 1.2 设计价值观

```
┌─────────────────────────────────────────────────────────────┐
│                     ExtHelper Design                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   简洁 ─── 最小化显示，只保留图标+名称                      │
│           ↓                                                │
│   高效 ─── 点击图标切换，右键菜单操作                      │
│           ↓                                                │
│   直观 ─── 分组水平展示，弹窗查看详情                      │
│           ↓                                                │
│   兼容 ─── 适配多种浏览器，保持体验一致                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 3. 组件设计

### 3.1 扩展卡片 - 卡片模式（列表视图）

```
┌─────────────────────────────────────────────────────────────┐
│  ┌────┐  Extension Name                                    │
│  │ 📷 │  Description text here...                         │
│  └────┘  v1.2.3                                    [开关] │
└─────────────────────────────────────────────────────────────┘
```

**设计细节**：
- 图标（48x48）+ 名称 + 描述 + 版本号
- 右侧显示启用/禁用开关
- 水平布局，单列显示
- 右键显示操作菜单
- 点击开关可切换启用/禁用状态

### 3.2 扩展卡片 - 紧凑模式（网格视图）

```
┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
│   📷   │ │   📷   │ │   📷   │ │   📷   │
│  Name  │ │  Name  │ │  Name  │ │  Name  │
└─────────┘ └─────────┘ └─────────┘ └─────────┘
```

**设计细节**：
- 固定卡片大小：130x120px
- 图标（48x48）+ 名称垂直排列
- 名称超出时省略（截断）
- 4列网格布局
- 点击卡片可切换启用/禁用状态

### 3.3 分组标签

```
┌─────────────────────────────────────────────────────────────┐
│  [● 工作 4 ⏻]  [● 开发工具 6 ⏻]  [+新建]               │
└─────────────────────────────────────────────────────────────┘
```

**设计细节**：
- 圆形标签设计
- 颜色点 + 分组名 + 数量
- 右侧小型电源按钮 ⏻ 用于一键启用/禁用分组内所有扩展
- 点击分组标签打开详情弹窗
- 无选中态高亮

## 4. 交互设计

### 4.1 核心交互

| 操作 | 触发方式 | 结果 |
|------|----------|------|
| 切换启用/禁用 | 点击开关（卡片模式）/ 点击卡片（紧凑模式） | 立即切换状态 |
| 打开分组详情 | 点击分组标签 | 打开分组详情弹窗 |
| 启用/禁用全组 | 点击分组右侧电源按钮 | 一键切换分组内所有扩展 |
| 更多操作 | 右键点击卡片 | 显示菜单 |
| 关闭弹窗 | ESC / 点击遮罩 | 关闭弹窗 |
| 切换视图 | 点击Header按钮（网格/列表图标） | 切换卡片/紧凑模式 |

### 4.2 动画规范

| 动画 | 时长 | 用途 |
|------|------|------|
| fade-in | 150ms | 显示弹窗/菜单 |
| fade-out | 100ms | 隐藏弹窗/菜单 |
