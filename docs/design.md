# UI/UX 设计文档

## 1. 设计原则

### 1.1 核心原则

1. **简洁优先** - 只显示必要信息，其他通过交互获取
2. **高效操作** - 减少点击次数，右键菜单整合所有功能
3. **一致性** - 统一的视觉语言和交互模式
4. **可访问性** - 符合WCAG 2.1 AA标准

### 1.2 设计价值观

```
┌─────────────────────────────────────────────────────────────┐
│                     ExtHelper Design                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   简洁 ─── 最小化显示，只保留图标+名称                      │
│           ↓                                                │
│   高效 ─── 点击图标切换，右键菜单操作                      │
│           ↓                                                │
│   直观 ─── 分组水平展示，弹窗查看详情                      │
│           ↓                                                │
│   兼容 ─── 适配多种浏览器，保持体验一致                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 4. 交互设计

### 4.1 核心交互

| 操作 | 触发方式 | 结果 |
|------|----------|------|
| 切换启用/禁用 | 点击图标 | 立即切换状态 |
| 查看分组详情 | 点击分组标签 | 打开弹窗 |
| 更多操作 | 右键点击卡片 | 显示菜单 |
| 关闭弹窗 | ESC / 点击遮罩 | 关闭弹窗 |
| 切换视图 | 点击Header按钮 | 切换卡片/紧凑模式 |

### 4.2 动画规范

| 动画 | 时长 | 用途 |
|------|------|------|
| fade-in | 150ms | 显示弹窗/菜单 |
| fade-out | 100ms | 隐藏弹窗/菜单 |
