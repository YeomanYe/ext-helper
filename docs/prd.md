# 产品需求文档 (PRD)

## ExtHelper - 浏览器扩展管理插件

---

## 1. 产品概述

### 1.1 产品背景

随着浏览器扩展生态的繁荣，用户安装的扩展数量越来越多。浏览器自带的扩展管理页面功能有限，无法满足以下需求：

- **快速查找** - 大量扩展时难以快速定位
- **分类管理** - 缺少灵活的分组成管理功能
- **快捷操作** - 需要多次点击才能完成简单操作

### 1.2 产品目标

构建一个**简洁、高效、跨浏览器**的扩展管理插件，帮助用户：

1. 快速管理和切换扩展状态（点击图标）
2. 通过分组整理扩展（水平标签+弹窗）
3. 提升日常工作效率

### 1.3 产品定位

| 维度 | 描述 |
|------|------|
| 产品类型 | 浏览器扩展插件 |
| 核心功能 | 扩展启用/禁用，分组管理 |
| 目标用户 | 浏览器扩展高级用户、开发者 |
| 浏览器支持 | Chrome、Firefox、Safari、Edge |

## 2. 用户故事

### 2.1 核心用户故事

```
作为用户，我想要：
1. 在一个界面查看所有已安装的扩展
2. 点击图标快速启用或禁用扩展
3. 通过分组标签快速查看某类扩展
4. 通过搜索快速找到目标扩展
5. 通过右键菜单进行更多操作
```

## 3. 功能需求

### 3.1 核心功能 (P0)

#### F1: 扩展列表展示

```
功能描述：展示用户安装的所有浏览器扩展

需求详情：
- 只显示扩展图标和名称（简洁设计）
- 支持卡片模式和紧凑模式
- 网格布局展示

交互：
- 点击图标 → 切换启用/禁用状态
- 右键点击 → 显示操作菜单
```

#### F2: 启用/禁用扩展

```
功能描述：快速切换扩展的启用状态

需求详情：
- 点击扩展图标即可切换状态
- 乐观更新，响应即时
- 状态持久化到浏览器

交互：
- 点击图标 → 立即切换 → 视觉反馈
```

#### F3: 搜索扩展

```
功能描述：通过名称搜索扩展

需求详情：
- 实时搜索，毫秒级响应
- 搜索扩展名称
- 清空搜索按钮
- 无结果显示空状态
```

#### F4: 分组管理

```
功能描述：管理扩展分组

需求详情：
- 水平分组标签栏
- 颜色标记分组
- 点击标签打开弹窗显示组内扩展

交互：
- 点击分组标签 → 打开弹窗显示组内扩展
- 点击"新建分组" → 创建新分组
```

#### F5: 快速筛选

```
功能描述：快速筛选扩展

筛选条件：
- 全部
- 已启用
- 已禁用
```

#### F6: 视图切换

```
功能描述：切换扩展展示方式

需求详情：
- 卡片模式：2列网格，显示更多信息
- 紧凑模式：1列列表，节省空间

交互：
- 点击Header按钮切换
- 状态持久化
```

### 3.2 增强功能 (P1)

#### F7: 右键菜单

```
功能描述：扩展操作菜单

菜单项：
- 启用/禁用
- 打开设置页面（如果有）
- 卸载扩展
```

#### F8: 扩展设置和卸载

```
功能描述：打开扩展设置页面和卸载

需求详情：
- 设置页面 → 打开扩展选项页
- 卸载 → 卸载扩展
```

## 4. UI设计规范

### 4.1 页面布局

```
┌────────────────────────────────────────┐
│  Header (Logo + 视图切换 + 设置)      │
├────────────────────────────────────────┤
│  SearchBar (搜索框)                    │
├────────────────────────────────────────┤
│  GroupChips (水平分组标签)             │
├────────────────────────────────────────┤
│  QuickFilters (筛选)                  │
├────────────────────────────────────────┤
│  Extension Grid (扩展卡片网格)          │
├────────────────────────────────────────┤
│  Footer (统计信息)                      │
└────────────────────────────────────────┘
```

### 4.2 交互规范

| 操作 | 触发 | 结果 |
|------|------|------|
| 切换状态 | 点击图标 | 启用/禁用切换 |
| 更多操作 | 右键菜单 | 显示操作列表 |
| 查看分组 | 点击标签 | 打开弹窗 |
| 关闭弹窗 | ESC/遮罩 | 关闭弹窗 |
| 切换视图 | Header按钮 | 卡片/紧凑切换 |

## 5. 非功能需求

### 5.1 性能需求

| 指标 | 目标 |
|------|------|
| Popup打开时间 | < 200ms |
| 扩展列表加载 | < 500ms |
| 搜索响应 | < 50ms |
| 状态切换 | < 100ms (乐观更新) |

### 5.2 兼容性需求

| 浏览器 | 最低版本 |
|--------|----------|
| Chrome | 120+ |
| Firefox | 121+ |
| Safari | 17.0+ |
| Edge | 120+ |

### 5.3 安全需求

- 最小权限原则 (management, storage)
- 本地存储优先
- 不收集用户数据
