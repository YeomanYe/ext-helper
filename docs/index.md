# 工程规范总路由

本目录是 ext-helper 工程规范的**分域索引**。总入口在 [`../CONTRIBUTING.md`](../CONTRIBUTING.md)。
每个域目录都有 `index.md`（导航）+ `rules.md`（该域硬规则总纲）。

## 按"你现在要干什么"找

| 任务 | 域 | 入口 |
|---|---|---|
| 改架构 / 数据流 / 加 store / repository | **architecture** | [architecture/](architecture/index.md) |
| 写组件 / 改业务代码 / 遵守编码约定 | **coding** | [coding/](coding/index.md) |
| 改样式 / punk 主题 / 设计 token | **ui** | [ui/](ui/index.md) |
| 改扩展运行时 / 权限 / manifest / 跨浏览器 | **extension** | [extension/](extension/index.md) |

## 非规范类文档（产品 / 设计 / 流程 / 历史，参考用）

- [prd.md](prd.md) — 产品需求
- [punk-design/](punk-design/) — 视觉主题文件（设计规则与设计稿在 [ui 域](ui/index.md)）
- [spec/](spec/) — 功能 spec 流程
- [_archive/](_archive/) — 历史文档：旧版完整架构/模块文档（`legacy/`）、Plasmo 迁移记录（`plasmo-migration/`）
