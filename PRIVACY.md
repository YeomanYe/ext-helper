# Privacy Policy for Ext Helper

**Last updated: April 19, 2026**

## Overview

Ext Helper ("the Extension") is a browser extension manager for Chrome, Firefox, and Edge. This privacy policy explains our data practices.

## Data Collection

**Ext Helper does not collect, transmit, or share any user data.**

Specifically, the Extension:

- Does **not** collect personal information
- Does **not** transmit any data to external servers
- Does **not** track browsing history or behavior
- Does **not** share data with third parties

## Data Stored Locally

The Extension stores the following data **locally on your device only**, using the browser's built-in storage API:

- Extension groups you create
- Automation rules you configure
- UI preferences (theme, view mode)
- Bisect session state

This data never leaves your device and is only used to provide the Extension's functionality.

## Permissions Explanation

| Permission | Purpose |
|---|---|
| `management` | Read and control installed extensions |
| `storage` | Save groups, rules, and preferences locally |
| `tabs` | Detect active tab URL to trigger automation rules |
| `alarms` | Schedule time-based automation rules |
| `host_permissions (<all_urls>)` | Read active tab URL for domain-matching rules only |
